利用Hexo+Github搭建个人博客

写这篇博客是帮助别的小伙伴想写个人博客所写，如有错误，请大神指出，欢迎留言，，，，< ^^>

安装Git Bash

一路点next就好了

下载地址

安装Node Js

一路点next就好了，注意在安装时Custom Setup选Add to PATH，这样不会再去配置电脑环境变量了，然后在Git中输入以下代码，查看环境变量是否配置。
node -v

安装Hexo

安装时先在自己盘符中新建一个文件夹，这个文件夹就是你本地博客的目录了，然后跳到这个文件夹打开Git Bash

cd 你的磁盘:/文件夹名

然后安装命令：

npm i -g hexo

初始化Hexo

初始化命令：

  1. hexo init

解释一下文件里面都是干什么的：

node_modules：是依赖包
public：存放的是生成的页面
scaffolds：命令生成文章等的模板
source：用命令创建的各种文章
themes：主题
_config.yml：整个博客的配置
db.json：source解析所得到的
package.json：项目所需模块项目的配置信息

Github设置

先去Github注册账号，没有的点这里

创建Github仓库

创建一个新的仓库

名称格式为 你创建的名字.github.io 按照这个规则创建

获取SSH

按照上面步骤创建好后，继续打开Git Bash
在Git Bash中配置github账户信息（youName和yourEail都替换成你自己的)
git config --global user.name "YourName"
git config --global user.email "YourEail"

创建ssh

ssh-keygen -t rsa -C "youremail@example.com"

一路点回车就行了，按照下图代码找到ssh

cd ~/.ssh
ls
cat id_rsa.pub

id_rsa.pub文件就是用来存放你SSH的

将SSH添加到Github

回到GitHub，点击头像进入设置（看不懂的浏览器有翻译）

找到SSH and GPG keys点击右上角的绿色的New SSH key

进去后Title随便填,Key就填你前面找到的SSH,然后点击Add SSH key就OK了

然后在Github中验证是否添加成功：

ssh -T git@github.com

如果出现以下界面，说明你安装成功了

恭喜你，可以蹦跶了！

发布博文

把上面的步骤弄完后，我们就可以发布我们的博文了

本地预览

现在githun中输入以下内容

npm i hexo-server

hexo 3.0把服务器独立成个别模块，需要单独安装,安装完成后再Git中再逐条输入以下内容:

hexo clean
hexo generate
hexo server

也可以简写为：

hexo clean
hexo g
hexo s

打开浏览器输入：http://localhost:4000

然后你就可以看见你想要的东西咯！…^^…

发布到Github

前面的是在本地预览，想要别人也看得见就得发布到Github上

先安装插件：

npm install hexo-deployer-git --save

然后去修改本地文件夹_config.yml里的内容

deploy:
type: git
  repo: https://github.com/YourgithubName/YourgithubName.github.io.git
 branch: master

地址改成你创建的用户名就行了

最后再次执行这个步骤就可以了

hexo clean
hexo generate
hexo deploy

ps：执行过程中会让你输入你的用户名和密码。

以后发布新的博文，都执行一下上面的步骤。

在浏览器中输入http://yourgithubname.github.io就可以看到你的个人博客啦，是不是很兴奋！如果你Github库的名字没有按照yourgithubname.github.io来起的话,就看不见页面了.

启用主题

自己的博客总得有个好看的主题吧！下面说说怎样去启用主题

主题可以去Hexo这里下载，具体方法如下：

选一个自己喜欢的主题点进去

选择这个地方是克隆或下载

先在Git中输入如下代码：

git clone https://github.com/iissnan/hexo-theme-next themes/next

后面的地址是刚刚克隆页面的那个地址，直接复制就好了

等主题下载好了后，在本地文件夹中修改站点配置文件_config.yml

将theme后面改成自己定的名字（就是将主题下载下来的那个名字可以直接改）

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: next

把这些弄好后，git继续重复这个步骤

hexo clean
hexo g
hexo s

打开浏览器输入：http://localhost:4000

就可以看见你的主题了，开心吧！

给博客添加个人域名

我是在腾讯上卖的域名传送门

域名解析

登录后进入右上角管理中心

修改域名DNS

注：需要修改域名DNS为：f1g1ns1.dnspod.net f1g1ns2.dnspod.net 立即修改

解析后需要等待一段时间才可生效（有快有慢）

添加CHAME文件

最后在你的hexo博客目录里的themes目录下的source目录下添加一个CNAME文件，不带后缀，内容如下[写下你自己的域名]：

//生成
hexo g
//push
hexo d

这样你就可以用自己额域名了

ps：这里说一下，域名得自己买（国内的有活动像腾讯的新用户可以一元购买一年挺划算）还有就是域名要认证。

最后大家可以看看我的博客主页 传送门

Hexo 使用说明

/新建文章
hexo n 文章名
//新建单页面
hexo n page 页面名
//重新生成
hexo g
//提交
hexo d
//生成+提交
hexo g -d
//备份
hexo b
//清空生成内容
hexo clean
//默认启动服务在4000端口
hexo s
